# Time Series Forecasting: Impact of Target Transformations

## Описание проекта

В данном проекте исследуется влияние различных преобразований целевой переменной на качество прогнозирования временных рядов.

Гипотеза исследования:

> Преобразования target (log1p, Box-Cox, differencing) улучшают качество прогноза не для всех временных рядов одинаково, а зависят от структуры ряда: тренда, сезонности и вариативности.

---

## Используемые методы

### Кластеризация временных рядов
- Извлечение признаков рядов
- Кластеризация (например, TimeSeriesKMeans / tsfeatures)
- Анализ типов рядов

### Бейзлайны
- Naive
- Seasonal Naive
- AutoETS
- AutoTheta

Важно: простые методы часто оказываются конкурентоспособными и служат хорошим baseline :contentReference[oaicite:3]{index=3}  

### Глобальная модель
- CatBoost

---

## Исследуемые преобразования

Для каждого временного ряда сравнивались:

1. **Без преобразований (original)**
2. **log1p**
3. **Box-Cox**
4. **Differencing**

Теоретически:
- log / Box-Cox → стабилизация дисперсии  
- differencing → устранение тренда и приведение к стационарности :contentReference[oaicite:4]{index=4}  

---

## Метрики

- SMAPE
- MAE

## Результаты

| Преобразование | MAE | SMAPE |
|----------------|-----|-------|
| original       | 635 | 0.156 |
| log1p          | 631 | 0.156 |
| box-cox        | - | 0.486 |
| differencing   | **576** | **0.146** |

---

## Выводы

- Differencing улучшает MAE, но не всегда SMAPE
- log1p даёт небольшое улучшение, но незначительное
- Box-Cox показал нестабильный результат
- Ни одно преобразование не превзошло baseline по SMAPE

Главный вывод:
> Преобразования полезны, но их эффективность зависит от структуры временного ряда.
